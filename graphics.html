mapgen

JavaScript | three.js | WebGL

An in-browser tool to create and edit
rock formations using voxel-aligned
tiles. Responsible for all aspects.

Try it online here (click to add a block,
shift+click to remove one, click and
drag to control the camera). 

https://github.com/ebogo1/mapgen

path tracer

C++

Implemented the path tracer following
Physically Based Rendering, including
BSDFs for diffuse, specular, dielectric,
and microfacet materials, as well as
several different rendering integrators.

Added functionality for acceleration
structures (KD-tree photon mapping),
implicit surfaces, constructive solid 
geometry, and environment mapping.

environment map: Paul Bourke

3D erosion model

C++ | OpenGL | GLSL

Implemented two erosion models
from this paper by Jacob Olsen.

Accelerated the iterative thermal
erosion algorithm by scaling
material distribution according to
the terrain's mean height gradient.

https://github.com/ebogo1/erosion-sim

mini minecraft

C++ | OpenGL | GLSL

Recreated Minecraft in a group of three.

Responsible for efficient terrain loading,
VBO management, OpenGL texturing
(per-block specularity, transparency,
animation), and a distance fog shader.
